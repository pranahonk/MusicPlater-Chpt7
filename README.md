## Run Locally

Clone the project

```bash
  git clone https://github.com/pranahonk/MusicPlater-Chpt7
```

Go to the project directory

```bash
  cd MusicPlater-Chpt7
```

Open Music Player

```bash
  open Music Player.xcodeproj
```
